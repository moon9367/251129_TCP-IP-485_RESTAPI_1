================================================================================
스마트팜 센서 데이터 수집기
================================================================================

## 개요
10초마다 센서 데이터를 수집하여 1분 평균값을 CSV로 저장하는 프로그램

## 수집 센서 (25개)
1. 내부온도
2. 내부습도
3. 내부일사량
4. 내부함수율
5. 내부수분장력
6. 외부온도
7. 외부습도
8. 외부일사량
9. 외부풍향
10. 외부풍속
11. 감우센서
12~25. 워드 65번 출력 상태 (비트 15~2, 각각 0 또는 1)
    - 유동팬습도조건출력중
    - 유동팬온도조건출력중
    - 차광열림출력
    - 차광닫힘출력
    - 상부보온열림출력
    - 상부보온닫힘출력
    - 천장우단열림출력
    - 천장우열림출력
    - 천장좌단열림출력
    - 천장좌열림출력
    - 제습출력
    - 난방출력
    - 관수출력
    - 유동팬출력

## 설치 방법
1. Python 3.7 이상 필요
2. 의존성 설치:
   pip install -r requirements.txt

## 실행 방법
run.bat 파일을 더블클릭하거나
명령줄에서: python temp_sensor_collector.py

## 저장 형식
- 파일명: @YYYY-MM-DD.csv (예: @2025-12-28.csv)
- 저장 주기: 매 1분마다 (절대시간 기준)
- 저장 위치: 실행 폴더 내

## 종료 방법
Ctrl+C 키 입력 (종료 전 남은 데이터 자동 저장)

## 포함 파일
- temp_sensor_collector.py : 메인 프로그램
- modbus_tcp_controller.py : Modbus 통신 모듈
- control_specs.py : 센서 제어 명세
- run.bat : 실행 배치 파일
- requirements.txt : Python 패키지 목록
